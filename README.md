# 糕点

#### 介绍

糕点 -- 第三方糕糕客户端

在阅读该项目时你需要入门以下知识：

Flask PyQt5 jQuery HTML CSS

#### 软件架构
软件架构说明

```yaml
- gaogao            #  根目录 糕点
```

#### 安装教程

由于项目使用了新特性联合类型

Python环境必须>=3.10 

1.  `pip install -r requirements.txt`
2.  `python app.py`
3.  `python main.py`
