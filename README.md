## 使用说明

由于项目使用了新特性联合类型

并且html,css,js的文件是存储在服务端的

**main.py** 仅供参考,主要代码为前端那边，py脚本仅用于修改hosts和打开网址

Python环境必须>=3.10 

1.  `pip install -r requirements.txt`
2.  `python main.py`
